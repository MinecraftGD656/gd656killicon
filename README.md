# 概述
模组发布页：https://www.mcmod.cn/class/21672.html

本模组为Minecraft添加了多种游戏风格的击杀图标与字幕，同时保留了极高的可配置性与可拓展性。

此文档最后编辑时间为 **2026年3月23日**，部分功能可能尚未提及或已被更改！请注意甄别。

由于 **1.1.0 版本模组代码完全重写**，故此文档部分内容将不再适用于 1.0.9 以下的版本！

## 兼容性列表
- 本模组不会对 Optifine 高清修复进行特别兼容；
- 本模组 1.0.9 及以下版本会与 AutoHUD 冲突 [出现轻微的多层图标]；
- 本模组 1.0.9 及以下版本会与 AdvancedCompass 冲突 [出现多层图标]；
- 本模组 1.0.4 及以上至 1.0.9 以下测试版需要 YetAnotherConfigLib 作为强制前置 [测试版内容不代表正式版功能，正式版中将不再使用本前置]；
- 本模组在 1.0.6 及以上版本支持**永恒枪械工坊：零**作为可选联动模组 [精确命中，末弹酬勇，火力压制等多个加分项与击杀类型]；
- 本模组在 1.0.9 及以上版本支持**卓越前线**作为可选联动模组 [精确命中，载具摧毁，载具修理等多个加分项与击杀类型]；
- 本模组在 1.0.9 及以上版本支持**永无止境：载具**作为可选联动模组 [载具伤害，载具摧毁，载具修理等多个加分项与击杀类型]；
- 本模组在 1.0.9 及以上版本支持**沉浸式飞机**作为可选联动模组 [载具伤害，载具摧毁，载具修理等多个加分项与击杀类型]；
- 本模组在 1.0.9 及以上版本支持**LR战术工坊**作为可选联动模组 [无赌而中加分项的闪光弹效果拓展]；
- 本模组在 1.0.9 及以上版本支持**索敌**作为可选联动模组 [索敌，标记击杀与标记小队助攻加分项]；
- 本模组在 1.1.0 及以上版本支持**Ping Wheel**作为可选联动模组 [索敌，标记击杀与标记小队助攻加分项]；
- 本模组在 1.1.1 及以上版本支持**自定义NPC非官方版**作为可选兼容模组 [击杀自定义NPC显示名称修复]；
- 对于服务端未安装本模组、仅客户端安装的情况下，本模组所有功能无效。

## 常见问题解答
- 高版本（1.21.x）Fabric 与 NeoForge：作者未放弃 Fabric 与 NeoForge 版本开发，主分支目前仍处于 beta 测试阶段（截至文档编辑时间），待测试完善后将会建立 fabric 与 neoforge 分支；
- 如何切换图标模式与配置：打开游戏内配置界面即可修改模式，内置教程指引；若无教程指引，可输入指令 `/gd656killicon client debug iamanew` 后再次打开配置界面，跟随教程修改图标配置。教程系统在 1.1.1.001 版本已完全完善，建议升级。

# 基本功能
> 以下内容在 1.0.9 版本之前不适用

- 本模组以预设为基础，提供多个官方预设：经典、CF连杀模式、CS2卡牌图标模式、Battlefield 1 模式、Battlefield 4 模式、Battlefield 5 模式、绝地求生模式、三角洲行动全面战场模式、无畏契约模式（含20+图标样式）。每个预设包含对应材质、音效和配置文件，玩家可自由切换、创建自定义预设，并导出预设为 gdpack 文件直接分享；
- 预设由元素构成，模组提供九个可配置元素，支持高度自定义；
- 每个预设包含一组音效，可在音效配置界面选中音效，直接拖入 OGG 或 WAV 文件快速自定义；
- 每个 killicon 元素拥有多种样式，可直接拖入 PNG 图片配置图标材质；拖入 GIF 文件会自动转为 PNG 长图并配置动画参数；
- 客户端会记录生涯数据：最远击杀距离、总击杀、宿敌玩家等；
- 服务端与客户端严格分离，客户端功能不影响服务端；
- 服务端核心为分数机制，管理员可自由修改玩家分数数据；
- 服务器会记录玩家统计信息：分数、死亡数、助攻数、击杀数，客户端记分板可查看全服玩家分数与 KDA；
- 服务端加分项为非管理员唯一加分方式，截至 1.1.0.013 版本共提供 **51 个加分项**，例如：火力压制、绝对制空权、勇者归来、马枪怪、标记小队助攻、背刺等，管理员可自定义每个加分项的加分表达式。

更多显示效果与样式以发布页 https://www.mcmod.cn/class/21672.html 相关视频介绍为准。
